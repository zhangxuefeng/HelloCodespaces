集合 $\{e^{(n)}:n\in\mathbf{N}\}$ 是闭集的证明是简单的,因为该集合里的任意两个不同元素的距离都是2,而且每个元素与自己的距离都是1,实际上类似于离散度量.其实集合 $\{e^{(n)}:n\in\mathbf{N}\}$ 既是开集也是闭集.
